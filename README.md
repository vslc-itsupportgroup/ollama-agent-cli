# Ollama CLI Agent

## Overview
The Ollama CLI Agent is a command-line interface (CLI) tool that enables users to interact with the Ollama server as an AI agent. This project provides a simple and efficient way to leverage AI capabilities directly from the terminal.

## Features
- Initialize and manage an AI agent.
- Execute commands and interact with the Ollama server.
- Easy-to-use CLI for seamless user experience.

## Installation
To install the Ollama CLI Agent, clone the repository and install the dependencies:

```bash
git clone https://github.com/yourusername/ollama-cli-agent.git
cd ollama-cli-agent
npm install
```

## Usage
To run the CLI agent, use the following command:

```bash
npx ollama-cli-agent
```

You can then enter commands to interact with the AI agent.

## Commands
The CLI supports various commands that can be executed. Refer to the documentation for specific command usage and examples.

## Running Tests
To run the tests for the project, use the following command:

```bash
npm test
```
